# Student-Record-Management-System-With-Exception

This project implements a complete **Student Record Management System** using:

- Object-Oriented Programming  
- Abstract Classes & Interfaces  
- Exception Handling  
- Custom Exceptions  
- File Handling  
- Multithreading  
- Java Collections  
- Sorting & Iterators  

---

## Features  
✔ Add, delete, update, search, view students  
✔ Multithreading-based loading simulation  
✔ Persistent storage using `BufferedReader` & `BufferedWriter`  
✔ Sorting by marks (descending order)  
✔ Iterator-based display  
✔ Custom exception: `StudentNotFoundException`  
✔ Follows modular structure using packages
